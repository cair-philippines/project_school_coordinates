# Reference — External Reference Data

External reference data that is neither DepEd-raw nor produced by our pipeline, but required for validation and enrichment. Not committed (large, externally sourced).

| Directory | Description | Source |
|---|---|---|
| `phl_admbnda_adm4_updated/` | Philippine barangay-level shapefile with PSGC codes (5 files: `.shp`, `.dbf`, `.shx`, `.prj`, `.cpg`). Currently reflects **Q4 2025 PSGC** boundaries. Used by `modules/validate_psgc.py` for point-in-polygon spatial validation. | Generated by `cair-philippines/open-data-philippine-maps` update pipeline, starting from the PSA/NAMRIA 2023 base shapefile |

## Shapefile refresh

PSA publishes PSGC updates quarterly. To refresh:

1. Pull the latest `cair-philippines/open-data-philippine-maps`
2. Run its `scripts/update_shapefiles.py`
3. Copy the resulting `out/phl_admbnda_adm4_updated.*` files here
4. Rerun the pipelines — the shapefile is read at validation time and does not require any code change

See `documentation/open_data_philippine_maps_feedback.md` for notes on that upstream repo.
